template<Integer T>
generator< short > print_ip_ns::print_ip_helpers::process ( const T &  addr)

Returns elements of addr like it's an ip address.

If T is an integer, returns addr's bytes in reverse order, starting from higher byte. If T is std::vector<t> or std::list<t>, where t is an integer, returns elements of T one by one.
